##### 懒渲染

html --> js --> fetch --> render 3*RTT 

优化3*RTT 👇

方案一：服务端渲染

方案二：server-push 客户端请求 html 时，服务端可预测出后续请求的 js/fetch 一并返回 

方案二缺点：无法判断本地是否已经缓存过待加载的资源，push多余的资源，消耗不必要的流量（解决：缓存状态存session）
